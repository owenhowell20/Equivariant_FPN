
SO(2)-Equivarient-Models Vault:
This is a small repository that contains SO(2)-equivarient versions of some commonly used deep neural network layers for computer vision applications.

Current Model Classes Implemented:

1. Equivariant Feature Pyrimid Network
2. Equivariant ResNet
3. Equivariant Mask-RCNN
4. Equivarient Transformer
